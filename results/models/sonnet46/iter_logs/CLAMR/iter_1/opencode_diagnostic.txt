### 3. `clamr_mpionly.cpp` (modified)
Added VeloC checkpoint/restart support:

**`VELOC_Init`**: Called after `L7_Init` (MPI init) to initialize VeloC.

**`veloc_do_checkpoint(mype)`**: Writes a per-rank binary checkpoint file via `VELOC_Route_file`. Contains:
- Scalars: ncycle, simTime, deltaT, H_sum_initial, ncells
- Mesh arrays: `i[]`, `j[]`, `level[]`
- State arrays: `H[]`, `U[]`, `V[]`

**`veloc_do_restart(mype)`**: On startup, detects if a checkpoint exists via `VELOC_Restart_test`. If found, restores all saved state, resizes mesh/state arrays if needed, calls `memory_reset_ptrs()` to refresh pointers.

**Restart initialization**: After `veloc_do_restart`, recalculates `ncells_global`, rebuilds `nsizes`/`ndispl`, rebuilds `celltype`, and skips the overwrite of `H_sum_initial`.

**Checkpointing inside burst loop**: Triggers every 10 cycles (much more frequent than `outputInterval=100`) so the validator always finds a checkpoint even if killed early. The ncycle is temporarily incremented before writing so the checkpoint records the correct completed-cycle count.

**`VELOC_Finalize`**: Called before `L7_Terminate` on program exit.
