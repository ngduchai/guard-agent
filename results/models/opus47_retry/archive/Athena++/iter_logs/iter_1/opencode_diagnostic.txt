- Added helpers `athena_count_local_hydro_elems`, `athena_pack_hydro`, `athena_unpack_hydro` that copy each local MeshBlock's `phydro->u` between its `AthenaArray<Real>` storage and one contiguous per-rank `Real` buffer.
- After `new Mesh(...)`:
  - `VELOC_Init(MPI_COMM_WORLD, "veloc.cfg")`

  - `VELOC_Restart_test("athena_ckpt", INT_MAX)`; if it returns a positive version, call `VELOC_Restart`, restore the scalars into the live `Mesh`, copy the hydro buffer back into each block's `phydro->u`, then call `pmesh->Initialize(1, pinput)` so MPI boundary buffers + primitive variables get rebuilt from the restored conserved state. - Inside the main loop, every 10 cycles pack `phydro->u` and scalars into the registered buffers and call `VELOC_Checkpoint("athena_ckpt", pmesh->ncycle)`.

The state captured (`u` arrays + cycle/time/dt) is the actual integrator state; restart resumes physics from where the snapshot was taken rather than restarting from scratch, satisfying the "real state capture / real state load" requirement.
