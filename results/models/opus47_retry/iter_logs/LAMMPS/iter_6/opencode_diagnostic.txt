2. **Fix**: Added a binary side-state file `<persistent>/lmp_ckpt_state_v<N>.bin` that holds `(tag, image, x[3], v[3])` for every atom at full native IEEE-754 precision. 

3. **Checkpoint path** (`gather_state_to_rank0`): every rank packs its `nlocal` atoms into an `AtomRecord` array; `MPI_Gatherv` collects them to rank 0; rank 0 serialises them with a small sanity header and writes the file atomically (write-`.tmp`-then-rename). 4. **Restart path** (`restore_state_from_buf`): rank 0 reads the .bin file, broadcasts to all ranks; each rank builds a `tag → AtomRecord*` hash map, then iterates its local atoms (already placed by `read_data` based on degraded positions) and overwrites `atom->x`, `atom->v`, `atom->image` with the full-precision saved values.
