**Edge case 1: No patches on this rank at level 0**
This can happen in parallel. The `num_patches = 0` would be written, and the restart would read `num_patches_ckpt = 0` - correctly skipping the patch loop.

**Edge case 2: Ghost box size mismatch**
If the checkpoint was written with a different ghost cell width (shouldn't happen with same input), the size check `uval->getGhostBox().size() == box_size` would fail and we'd skip. This is safe.

**Edge case 3: VeloC checkpoint directory issues**
If VeloC can't write to `/tmp/samrai_veloc_scratch`, the checkpoints will fail. But the validator pre-creates these directories, so this should work.

**Edge case 4: Version number overflow**
With `max_integrator_steps = 300000` and `CKPT_INTERVAL = 1`, the version number goes up to 300000. `int` can hold up to ~2 billion, so no overflow.
