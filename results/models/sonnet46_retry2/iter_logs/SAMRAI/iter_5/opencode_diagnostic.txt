## Root Cause
The recovery run crashed with a segfault (exit code 139) inside `regridAllFinerLevels`. This function has complex internal state requirements (connector widths, tag allocation, level data availability) that weren't satisfied when called directly from the restart logic after removing all fine levels.

## Fix

**`source/SAMRAI/algs/TimeRefinementIntegrator.h`**: Added two new setter methods `setLevelDtMax()` and `setLevelDtActual()` to allow the restart code to properly initialize the integrator's timestep bookkeeping.

**`source/test/applications/LinAdv/main.cpp`**: Rewrote `restoreCheckpoint` to:
1. Only restore level 0 (coarsest) patch data from the checkpoint
2. Remove all fine levels (level 1+) without trying to regrid them
3. Set integrator time, step count, and timestep metadata to checkpoint values
4. **Remove the `regridAllFinerLevels` call entirely**

The SAMRAI `TimeRefinementIntegrator::advanceHierarchy()` naturally rebuilds fine levels at the first regrid point during the normal time-step loop (when `atRegridPoint()` returns true). This is the safe and correct approach that avoids all the complex state requirements of `regridAllFinerLevels`.
