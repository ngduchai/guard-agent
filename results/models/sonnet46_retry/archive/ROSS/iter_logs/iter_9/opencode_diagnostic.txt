## Fixes Applied

### `models/phold/phold.main.c`

1. **Added `all_ok()` helper** — wraps `MPI_Allreduce` with `MPI_LAND` to collectively decide go/no-go.

2. **`veloc_gvt_hook` (checkpoint)**:
   - Wall-clock timer decision now uses `all_ok(local_ready)` — if ANY rank's timer hasn't elapsed, ALL ranks skip.
   - OOM/overflow abort uses `all_ok(local_ok)` — if ANY rank can't collect events, ALL abort.
   - All ranks call `VELOC_Checkpoint_begin` → check with `all_ok` → either all proceed to `VELOC_Checkpoint_mem` or all call `VELOC_Checkpoint_end(0)`.
   - All ranks call `VELOC_Checkpoint_mem` → check with `all_ok` → either all write files or all call `VELOC_Checkpoint_end(0)`.
   - All ranks call `VELOC_Checkpoint_end` after agreeing on final success/failure.

3. **`veloc_post_lp_init` (restart)**:
   - Uses `g_veloc_version` (already broadcast from rank-0 in `main()`) instead of calling `VELOC_Restart_test` again.
   - ALL ranks call `VELOC_Restart_begin` → check with `all_ok` → either all proceed or all call `VELOC_Restart_end(0)`.
   - ALL ranks call `VELOC_Recover_mem` → check with `all_ok` → either all read files or all call `VELOC_Restart_end(0)`.
   - File read success agreed upon with `all_ok` before proceeding.
   - ALL ranks call `VELOC_Restart_end` at the end.

4. **`main()` checkpoint detection** — uses `MPI_Bcast` to share rank-0's checkpoint version so all ranks have the same `g_veloc_version`.
