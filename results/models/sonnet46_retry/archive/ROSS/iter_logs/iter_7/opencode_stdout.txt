The standard way to do iterative in-order traversal with parent pointers:

1. Start at minimum (leftmost = `pq->least`)
2. For each node: visit it, then find its in-order successor
3. In-order successor: if node has a right child, go right then leftmost. Otherwise, go up until we came from a left child.

```c
void tw_pq_traverse(tw_pq *pq, ...) {
    tw_event *cur = pq->least; // minimum element
    while (cur) {
        cb(cur, user);
        cur = inorder_successor(cur);
    }
}

static tw_event *inorder_successor(tw_event *node) {
    if (RIGHT(node)) {
        // go to rightmost's leftmost
        tw_event *s = RIGHT(node);
        while (LEFT(s)) s = LEFT(s);
        return s;
    }
    // go up until we came from a left child
    tw_event *p = UP(node);
    while (p && node == RIGHT(p)) {
        node = p;
        p = UP(p);
    }
    return p;
}
```
