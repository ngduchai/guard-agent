Add VeloC checkpoint/restart support to this SW4lite seismic wave propagation code. The main simulation loop is in src/EW.C. The code solves 3D elastic wave equations using finite differences on a structured grid with MPI parallelism.

Key state to checkpoint:
- The displacement arrays (Um, U) which are vectors of Sarray objects
- The simulation time and cycle count
- Any attenuation memory variables if enabled

Requirements:
1. Initialize VeloC after MPI_Init
2. Register critical state arrays with VELOC_Mem_protect
3. Add restart logic before the main time-stepping loop to check for existing checkpoints
4. Add periodic checkpointing inside the time-stepping loop
5. Finalize VeloC before MPI_Finalize
6. Create veloc.cfg with appropriate scratch/persistent paths
7. Update the Makefile to link against the VeloC client library
