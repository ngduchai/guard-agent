Add VeloC checkpoint/restart support to this VPIC particle-in-cell simulation code. VPIC simulates kinetic plasmas using the particle-in-cell method with MPI parallelism.

Key state to checkpoint:
- Particle data arrays (position, velocity, weight for each species)
- Field data (electric and magnetic fields on the grid)
- Simulation time step and cycle count
- Accumulator arrays

Requirements:
1. Initialize VeloC after MPI initialization
2. Register critical particle and field arrays with VELOC_Mem_protect
3. Add restart logic to check for existing checkpoints before the main simulation loop
4. Add periodic checkpointing inside the main time-stepping loop
5. Finalize VeloC before MPI_Finalize
6. Create veloc.cfg with appropriate scratch/persistent paths
7. Update CMakeLists.txt to find and link the VeloC client library
