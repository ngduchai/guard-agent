Add VeloC checkpoint/restart support to this Athena++ astrophysical MHD code. The code solves the equations of magnetohydrodynamics on a block-structured adaptive mesh with MPI parallelism.

Key state to checkpoint:
- Conserved variables (density, momentum, energy) on each MeshBlock
- Magnetic field components (if MHD enabled)
- Current simulation time and cycle number
- Mesh structure and refinement levels

Requirements:
1. Initialize VeloC after MPI_Init in main.cpp
2. Register MeshBlock data arrays with VELOC_Mem_protect
3. Add restart logic before the main time integration loop
4. Add periodic checkpointing inside the main loop (src/main.cpp)
5. Finalize VeloC before MPI_Finalize
6. Create veloc.cfg with appropriate scratch/persistent paths
7. Update the Makefile to link against the VeloC client library
