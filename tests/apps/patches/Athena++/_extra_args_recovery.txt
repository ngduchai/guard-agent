# Recovery-attempt args for Athena++ reference bench (attempt_2+).
# Athena++ requires explicit -r <file> to restart; no auto-detection
# (src/main.cpp:155-158, src/main.cpp:205-213).  Restart files are
# written as <problem_id>.<5-digit-counter>.rst.
#
# The patched athinput.blast (this dir) sets <output3> file_type=rst
# dt=5.0 with tlim=19.0 → file_number sequence:
#   Blast.00000.rst at t=0 (initial dump)
#   Blast.00001.rst at t=5
#   Blast.00002.rst at t=10
#   Blast.00003.rst at t=15
# Injection delay is ~50% of nominal runtime (~38s on a 76s baseline),
# which corresponds to roughly t=10 sim time → Blast.00001.rst is
# guaranteed to exist; Blast.00002.rst may also exist.  Pick the
# earlier-guaranteed file so attempt_2 always has something to read.
-r
Blast.00001.rst
